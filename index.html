HPE Synergy Software Releases - Overview

HPE Synergy Software Releases are available for customers to download and use on products which are under an active HPE Warranty or an HPE Support Agreement.

HPE Synergy Software Releases are comprised of HPE Synergy Management combinations (a versioned set of Composer and Image Streamer software) and HPE Synergy Custom SPPs. Combinations of specific releases have been developed, tested, and released together. For Composer and Image Streamer, combinations for download are provided: firmware updates to existing images, and complete images for recovery (in the event that re-imaging of the system is needed).

Attention

Updated System ROM firmware in HPE Synergy Custom SPPs 2019.03.20190826, 2019.03.20190825, 2019.03.20190718, 2019.03.20190612, 2019.03.20190514, and 2018.11.20190611 address potential issues of uncorrectable memory errors or boot time memory training issues with Intel Xeon Scalable Performance Bronze, Silver, and Gold 5x00 processors. (This issue is not unique to HPE servers.) HPE recommends that any customer experiencing a memory issue immediately update their system ROM before replacing any components. For more information, see Customer Notice a00072625en_us.

CAUTION: HPE Synergy Custom SPP 2018.11.20190205 has an identified issue as described in Customer Bulletin a00069625en_us. It has been replaced by HPE Synergy Custom SPP 2018.11.20190405.

HPE Composer (HPE OneView) 4.20.01 has an identified issue as described in Customer Advisory a00068248en_us. This issue is less likely to occur with HPE Synergy Composer because fewer Synergy systems have the required number of drives to trigger the issue. This HPE Composer (HPE OneView) 4.20.01.01 release corrects the issue.

IMPORTANT: Immediately before upgrading to HPE OneView 4.20.02, install HPE_PreUpgrade_4.20_5.00_Z7550-96608.bin. This file is accessible from the HPE Support Center download pages for the HPE Synergy Management combinations.

Upgrading to HPE Synergy 5.0

NOTE: The HPE Synergy Composer 5.0 Update file will be made available mid September.

Upgrading to HPE OneView 5.0 on an HPE Synergy Composer requires more time than earlier releases. Due to the additional time required to upgrade the HPE Synergy Composers for this release, it is permissible to defer updating HPE Synergy Image Streamer to a separate appliance maintenance window. Refer to the HPE OneView for Synergy and HPE Synergy Image Streamer release notes for specific update times and general upgrade requirements.

Any HPE OneView 4.10 or later can be upgraded to version 5.0.

For more details on these changes, review the HPE OneView 5.0 Release Notes for HPE Synergy and the HPE Synergy Image Streamer 5.0 Release Notes.

NOTE: During the upgrade, after the progress bar displays 100%, the browser will display Temporarily unavailable for several minutes while the post-upgrade refresh processing is taking place. For details, refer to Customer Advisory a00082207en.

HPE Passport account login and download

Access to the software releases requires validation via the HPE Support Center. You must be logged in and have a valid HPE warranty or HPE support agreement linked to your account.

  1. Log in to your HPE Passport account (one-time) (HPE Support Center).
  2. Link your HPE warranty or HPE support agreement to your HPE Passport account, if you have not already done so.

    NOTE: You only have to perform this linking process once.

  3. Select an HPE Synergy Management combination from the table above.

In future downloads, the HPE Support Center will recognize your entitlement as long as your product has an active HPE Warranty or HPE Support Agreement.

For instructions on deploying software updates, see the HPE OneView for HPE Synergy Firmware and Driver Update Guide on the Hewlett Packard Enterprise Information Library.

HPE Synergy Management combination of Composer 5.0 and Image Streamer 5.0

IMPORTANT: New hardware and features introduced for HPE Synergy 5.0 require the latest HPE Synergy Custom SPP.
New feature capabilities may depend on specific Synergy Custom SPP versions. See the HPE Synergy Firmware Feature Comparison Table on the HPE Synergy Software Release Information site.

This Synergy Management combination provides enhancements which include:

  • Virtual Connect Fabric & Networking support for Synergy IPv6 operations, Large network sets, and Arista Top-of-Rack (ToR) switches
  • HPE Synergy Composer2
    • iLO Remote Access for Composer2 management appliances
    • Enhanced security and performance with Secure boot and Silicon Root of Trust.
    • Expanded firmware repository.
  • Firmware Compliance Dashboard for Gen10-based servers
  • Consistency Reporting for Firmware & BIOS (Server Profile to Server Hardware)
  • Brocade REST API (FOS) for BNA replacement
  • Supportability and Remote Support enhancements
  • Cluster profile rolling cluster update for VMware vSAN
  • Image Streamer 5.0 enablement for:
    • Use with VC SE 100Gb F32 Module and Composer2
    • Pre-upgrade health checks and bundled support tools
    • OS Deployment artifacts for SLES 12 SP3 or SP4, RHEL 7.6, SLES OS with BTRFS (using FAT32), and RHEL OS with XFS (using FAT32)

HPE Synergy Custom SPP 2019.03.20190826

This Synergy Custom SPP is based on the March-2019 Core SPP 2019.03.1, and it provides:

  • Updated iLO5 firmware and System ROM firmware
  • Support for VMware ESXi 6.5 U3 and ESXi 6.7 U3
  • New Synergy product support:
    • HPE Virtual Connect SE 100Gb F32 Module for HPE Synergy version 2.0.0.1006
    • HPE Synergy 50Gb Interconnect Link Module (25/50Gb Satellite module) version 1.0.0.1002
    • HPE Virtual Connect SE 40Gb F8 Module for HPE Synergy version 1.5.0.1004
    • HPE Synergy 40Gb F8 Switch Module version 1.5.0.1004
    • HPE Virtual Connect SE 32Gb FC Module for HPE Synergy version 5.10.69.
    • HPE Virtual Connect SE 16Gb FC Module for HPE Synergy version 5.10.69
    • HPE Synergy 6820C 25/50Gb Converged Network Adapter
    • HPE Integrated Lights-Out (HPE iLO 5) V 1.46 Jul 25 2019
    • HPE Synergy 4-Port Frame Link Module
      • Hardware root of trust anchors essential firmware to the Frame Link Module processor.
      • Secure Start ensures firmware integrity for the Frame Link Module firmware.
      • TPM protects the key used for file encryption to safeguard Frame Link Module data.
  • For a complete description, see the Hewlett Packard Enterprise Support Center.
